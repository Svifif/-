# Расчет энтропии сложной системы

Задание N2: средство для расчета энтропии сложной системы, состоящей из двух зависимых систем.

## Запуск

Откройте файл `index.html` в любом современном web-браузере. Сборка и установка зависимостей не требуются.

## Входные данные

Матрица вероятностей совместных состояний `P(Ai, Bj)`. Все элементы должны быть неотрицательными, а сумма вероятностей должна быть равна `1`. При необходимости можно включить нормирование матрицы.

## Что рассчитывается

- `H(A,B)` — совместная энтропия сложной системы.
- `H(A)` — энтропия первой системы.
- `H(B)` — энтропия второй системы.
- `H(A|B)` — условная энтропия первой системы при известной второй.
- `H(B|A)` — условная энтропия второй системы при известной первой.
- `I(A;B)` — взаимная информация.

Все значения выводятся в битах.
